HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element defines metadata such as author, kewords, etc.
  • The style element defines the CSS styles associated with a website
  • The link element connects an external resource to the HTML document

CSS

  • CSS stands for Cascading Style Sheet, a set of rules that define how elements look/act in an HTML document.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The character "*" is used to style ALL HTML elements in the file.
  • CSS statements are formatted as: selector {declaration(property: value;) declaration(property:value)}.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls the latest main branch to the workspace
  • git clone: clones a repository to the workspace
  • GitHub allows companies/groups/individuals to manage versions of their projects/applications/products
